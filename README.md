# Cerebryx

Cerebryx is a modular neuromorphic system for designing, simulating, and eventually building brain-inspired distributed networks. It explores how cognition-like behavior can emerge from simple, locally connected units arranged in structured 3D topologies.

The project combines:
- A configurable 3D neural topology generator
- Cluster-based brain region simulation
- Gaussian and small-world connectivity models
- Interactive visualization of neurons, connections, and clusters
- A future path toward hardware mapping on RP2040 / ESP32 microcontroller networks

Cerebryx is not a traditional AI system. Instead, it is an exploration of structure-first intelligence: how geometry, connectivity, and local rules can give rise to emergent computation.

---

## Concept

In Cerebryx, a "brain" is not defined by a model or algorithm, but by structure:

- Neurons are simple nodes in space
- Clusters represent functional brain regions
- Connections follow biologically inspired probability distributions
- Behavior emerges from topology, not centralized logic

The goal is to study and design systems where intelligence is a property of architecture.

## Features (current & planned)

### Simulation
- 3D neural graph generation
- Clustered spatial organization
- Gaussian local connectivity
- Small-world long-range links
- Fully parameter-driven topology generation

### Visualization
- Interactive 3D rendering of neurons and connections
- Cluster visualization via spatial grouping
- Adjustable structure parameters in real time

### Planned
- Activity / spike simulation layer
- Learning via local weight adaptation (Hebbian / reinforcement)
- Export format for hardware mapping
- Multi-node microcontroller architecture (ESP32 / RP2040)
- CAN-bus based neural communication layer

## Architecture Vision

Cerebryx is designed as a bridge between:

- Biological neural organization
- Graph-based computation models
- Embedded distributed systems

Future hardware implementations aim to treat each microcontroller as a neural cluster, forming a scalable, physically distributed artificial nervous system.

## Core Idea

> Intelligence is not programmed — it emerges from structure.

The core idea behind Cerebryx originates from a broader hypothesis about consciousness and experience.
The assumption is that if a system becomes sufficiently complex to *experience* and process information in a structured, self-referential way, something akin to consciousness may emerge as a natural property of that system rather than as a designed feature. This leads to a speculative but guiding idea: that consciousness is not necessarily tied to biological matter, but may arise whenever information processing systems reach a certain level of complexity, recurrence, and internal connectivity.
In this view, information is not abstract or external, but locally bound within the system itself. Consciousness is then not a separate “thing”, but the act of experience — the continuous processing, propagation, and transformation of information within a sufficiently rich structure. Cerebryx is an experimental attempt to explore this idea in a concrete, engineering-driven way: by building a modular, spatially embedded network of simple computational units whose behavior emerges from topology, interaction, and feedback rather than explicit programming.
It is not a claim about what consciousness is, but an exploration of what kinds of structures might make “experience” possible.
